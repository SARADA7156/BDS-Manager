# Obsidian(サーバー管理中核部分)のコード一覧

**コード一覧**
| Code(整数) | カテゴリ       | 説明(メッセージ)                                     | 対処法/発生原因                                                                                                                  | 
| ---------- | -------------- | ---------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------- | 
| 200        | 成功           | 処理が正常に終了しました。                           | 正常終了                                                                                                                         | 
| 202        | 受理           | リクエストは受理されましたが、まだ完了していません。 | 別のインスタンスを作成中のため、リクエストが現在キューで待機中です。                                                             | 
| 400        | 入力エラー     | インスタンス設定データの形式が不正です。             | インスタンス設定のデータ形式が不正(例: レコードの数が足りないなど)な時に発生します。                                             | 
| 429        | リソース不足   | 作成可能な最大インスタンス数に到達した。             | 利用可能なポートがなく、最大インスタンス数に到達しています。                                                                     | 
| 500        | サーバーエラー | 汎用的なサーバーエラー                               | インスタンスの作成中に何らかの致命的なエラーが発生しました。                                                                     | 
| 504        | タイムアウト   | リソース取得時のタイムアウト                         | サーバーが外部サイトへリソースを要求したが、リクエストがタイムアウトしたため、後の処理を継続することができない場合に発生します。 | 
| 600        | セキュリティ   | 汎用セキュリティエラー                               | 詳細不明なセキュリティに関する違反/エラー                                                                                        | 
| 601        | セキュリティ   | リソースセキュリティ違反                             | 外部サイトから取得したデータの中に、セキュリティに反するデータ(不正なファイル名)が発生したため処理がロックされました。           | 
| 700        | ファイルI/O    | 汎用ファイルI/Oエラー                                | 詳細不明なファイルI/Oに関するエラー                                                                                              | 
| 701        | ファイルI/O    | ファイルが見つかりません                             | 指定されたパスにファイルやディレクトリが存在しない。                                                                             | 
| 702        | ファイルI/O    | アクセスが拒否されました                             | 必要な読み取り/書き込み権限が無い                                                                                                | 
| 703        | ファイルI/O    | ファイルパスが無効です。                             | OSの規定に反する不正なファイルパス(使用禁止文字など)。                                                                           | 
| 704        | ファイルI/O    | ディスク容量不足                                     | ファイルを書き込みための空き容量がない。                                                                                         | 
| 705        | ファイルI/O    | ファイル破損                                         | ファイルが破損し、その後の処理が継続できない。                                                                                   |
